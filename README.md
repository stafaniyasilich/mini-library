# Mini Library Project

This project is a simple Java application that simulates a library system.

## Features

- Book class (title, author, pages, availability)
- Reader class (library user)
- Library class (stores books in an array)
- Borrowing and returning books

## Classes

- Book
- Reader
- Library
- Main

## How to run

Run the Main class to see the test scenario.

## Scenario

- Add books to the library
- Print available books
- Borrow a book
- Return the book
